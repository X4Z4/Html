-Tipos de imagens

os tipos de imagens que são interessantes de ser usar em um site são: jpeg e png, sendo que o png, cria imagens marios mas com fundo transparente e o jpeg, cria imagens com maior compactação mas com menor qualidade e que não podem ficar com fundo transparente. o comando utilizado para colocar imagens em um site é (img src="" alt=""), onde src é onde se encontra o arquivo e o alt é o que sera escrito na tela caso a imagem não seja carregada

Vale citar que existem 3 jeitos de adicionar uma imagem a um site, sendo elas :

1-puxar a imagem da mesma onde se encontra o index.html

2-puxa de dentro de uma subpasta do index

3-puxar de um site de dentro da internet usando (img src="" alt="") e dentro do img vc põe endereço da imagem
